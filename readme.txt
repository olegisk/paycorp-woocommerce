== Description ==

Paycorp International (a wholly owned subsidiary of Bancstac) Payment Gateway for WooCommerce.
The plugin provides seamless PCI DSS certified payment processing for credit card payments.
Features:
* Credit Card payments through PayCorp
* Tokenization
* Recurring payments using WooCommerce Subscriptions
